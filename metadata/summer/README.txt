This directory contains the metadata associated with the summer portion of the "What Makes a Migrant Monarch?" project (June, 2023).

Directory: field_site
Description: Directory contains a map of how plants were distributed through the field site, as well as data logged by environmental sensors placed throughout the field site.

Directory: monarch_rearing
Description: Directory contains metadata pertaining to monarch rearing, genetic backgrounds, and monarch lineage-plant mappings.
