This directory contains metadata associated with monarchs reared in June during the WMMP. The contents of this document provide descriptions of the files in this directory, as well as descriptions of each column within tabular files.

File : lineages.tsv
	description : Data pertaining to the monarch butterflies that were placed near the field site at the start of the experiment (06-01-2023).
	column[individual-id] : Unique identifier for individual monarch butterflies available to start matings.
	column[sex] : The sex of the corresponding butterfly (M = male, F = female). 
	column[date-eclosed] : The date that the monarch eclosed and was placed in the refrigerator to induce a state of diapause.
	column[mating-cage] : The cage that the corresponding individual was placed in for mating.
